# Puckdoug Tap

## How do I install these formulae?

`brew install puckdoug/tap/<formula>`

Or `brew tap puckdoug/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "puckdoug/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
